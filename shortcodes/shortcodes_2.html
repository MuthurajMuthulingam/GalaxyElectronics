Buttons

Shortcode Options:

  • title=”…” (The title of the button)
  • color=”…” (Can be set with preset colors: ocean, coal, blush, violet or with any hex color value) Need a hex color generator?
  • type=”normal, normal-rounded, big, big-rounded”
  • position=”…” (Optional _ Can be filled with: left – right – top – bottom)
  • target=”…” (Optional _ Can be set with: “new” to open the link in a new page or “same” to load the page on current window)
  • url=”…” (Enter the destination url desired)

Normal Color Preset Buttons:
Ocean Coal Blush VioletSage Hazel Blaze SandOrange Blue Pink Green
[sp_button title="Default Color" color=ocean position=right target=same url=# type="normal"]

Normal-Rounded Color Preset Buttons:
Ocean Coal Blush VioletSage Hazel Blaze SandOrange Blue Pink Green
[sp_button title="Default Color" color=ocean position=right target=same url=# type="normal-rounded"]

Grid System

Flexible Columns allow you to build flexible columns with ease.

Shortcode options:

  • span=”…” (one_half, one_third, one_fourth, and two_third)
  • last=”…” (true – you want to set this to true whenever you are declaring the last or only column so the rest of the content will clear properly)

NOTE: You will see a space after the trailing forward slash on the closing shortcode tag. Remove that space when you’re using this shortcode. We had to add this space in our examples or else the grid system gets triggered.

Display Google Maps

Shortcode Options:

  • width=”…px or %” (Optional _ Set the width of the map in pixels. If left empty it will automatically set to 500px default width)
  • height=”…px or %” (Optional _ Set the height of the map in pixels. If left empty it will automatically set to 500px default height)
  • zoom=”…” (Optional _ Set the level of zoom desired (1-20), no % symbol needed)
  • address= … (Enter your desired address to display on the map)

 

[sp_map width=100% height=300px zoom=15 address="Dolores Park, San Francisco, CA"]

Lightbox

Shortcode Options:

  • title=”…” (Optional – Insert the title you want to display when the Lightbox Pops up)
  • url=”…” (Enter the url of the video source or image)
  • group=”…” (Use this option to group images in the same lighbox)
  • thumbnail=”…” (Choose between “large” and “small” to show the default video thumbnail)
  • enter in between [ sp_lightbox ... ] and [ /sp_lightbox ] the image or text you want to use to trigger the lightbox
  • slideshow=”…” (Optional – you can set “true” to enable slideshow for your images, default is “false”)
  • theme=”…” (Choose different themes for your lightbox “facebook,light_rounded,dark_rounded,light_square,dark_square. Default is “facebook”)
  • show_social=”…” (Enter “true” if you want to display social tools for your lightbox. Default is “false”)

Custom Youtube / Vimeo Thumbnails [sp_lightbox title="..." video url="..."] <img src="..."> or any text [/sp_lightbox]

Default Youtube / Vimeo Thumbnails [sp_lightbox thumbnail=small title="..." video url="..."]

Home / Theme Path

These two shortcode will automatically insert the path of the root domain name and template path to provide convenience so that you would not have any broken link or image if you transfer your site on another domain.
[sp_home]
is a shortcode to output the path of your site domain.
For example this [sp_home]image.png would be the equivalent to http://yourdomainname.com/image.png
[sp_theme]
is a shortcode to output the path to your themes folder.
For example this “[sp_theme]image.png” would be the equivalent to “http://yourdomainname.com/wp-content/themes/yourtheme/image.png”
